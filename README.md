# MoodMate AI 💙

MoodMate AI is an AI-powered emotion detection and music recommendation application.

## Features

- Detects user's emotion from text
- Uses Machine Learning for sentiment/emotion classification
- Provides confidence score
- Gives motivational quotes
- Recommends songs according to mood

## Tech Stack

### Machine Learning
- Python
- Scikit-learn
- NLP
- TF-IDF Vectorization
- Logistic Regression

### Backend
- Flask
- REST API

### Frontend
- React.js
- CSS

## Project Flow

User Text
↓
NLP Preprocessing
↓
ML Model Prediction
↓
Emotion Detection
↓
Song + Quote Recommendation
